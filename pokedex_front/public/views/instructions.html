INSTRUCCIONES PARA LA REALIZACIÓN DE LA PRUEBA

Descripción de la prueba:

Esta prueba tiene como objetivo analizar las aptitudes en los framework Django y Angular de los aspirantes al cargo.

Descripción del proyecto:

Este proyecto permite la creación y visualización de Pokemón y opcionalmente su edición y eliminación.

Criterios de evaluación:

  1. El buen uso de la herramienta GIT.
  2. La organización del código.
  3. La sintaxis utilizada para escribir el código en cada lenguaje.
  4. El uso correcto de las herramientas.

REQUERIMIENTOS MÍNIMOS:

  1. Tareas Backend
    • Se debe crear un WebService que permita crear un Pokemón.
    • Se debe crear un WebService que retorne la lista de los Pokemón almacenados en base de datos.
    • cada uno de los WebServices deberá tener sus respectivos tests que comprueben su correcto funcionamiento.
  2. Tareas Frontend
    • Se debe crear una View que permita la creación de un Pokemón consumiendo el WebService respectivo.
    • Se debe crear una View que permita listar los Pokemón consumiendo el WebService respectivo.

TAREAS OPCIONALES:

  1. Tareas Backend
    • Se debe crear un WebService que permita modificar un Pokemón.
    • Se debe crear un WebService que eliminar un Pokemón almacenado en base de datos.
    • cada uno de los WebServices deberá tener sus respectivos tests que comprueben su correcto funcionamiento.
  2. Tareas Frontend
    • Se debe crear una View que permita la edición de un Pokemón consumiendo el WebService respectivo.
    • Se debe crear una View que permita la eliminación de un Pokemón consumiendo el WebService respectivo.
